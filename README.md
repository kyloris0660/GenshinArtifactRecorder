# 原神圣遗物记录脚本

## 说明

将图像或视频中出现的圣遗物面板信息保存至excel文件中，用来记录我每天刷圣遗物本的掉落情况。

面板信息可以为背包中或副本结算界面，只要出现面板就行。

文字识别调用百度AI的api。
